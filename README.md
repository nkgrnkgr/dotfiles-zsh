# dotfiles-zsh

## require

```
brew install pure
brew install sheldon
brew install fzf
brew install ripgrep
brew install bat
brew install fd
brew install zoxide
brew install mise
```

## install

`git clone git@github.com:nkgrnkgr/dotfiles-zsh.git`

## create link

`ln -sf ~/dotfiles-zsh/.zshrc ~/.zshrc`

`ln -sf ~/dotfiles-zsh/mise/.config.toml ~/.config/mise/config.toml`
